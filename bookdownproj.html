3 Transect Report

3.1 Overview

This report provides summary statistics and figures for ongoing transect sampling. The first section of the report focuses on the current sampling (Winter 2021-2022) and how the collected data compare to last year’s sampling (Winter 2020-2021). So far 35 days have been sampled this season. The second half of the report gives summaries of all of the data that have been collected since the beginning of the project (5/27/2010). In total, 140 days have been sampled over this entire project.
